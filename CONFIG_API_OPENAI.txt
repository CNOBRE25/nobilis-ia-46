CONFIGURAÇÃO DA API OPENAI

Para que a IA funcione corretamente, você precisa configurar a API OpenAI:

1. Crie um arquivo .env.local na raiz do projeto
2. Adicione sua chave da OpenAI:

VITE_OPENAI_API_KEY=sua_chave_openai_aqui

3. Obtenha sua chave em: https://platform.openai.com/api-keys

4. Reinicie o servidor de desenvolvimento após criar o arquivo

NOTA: Se a API não estiver configurada, o sistema funcionará em modo de simulação.
